# fem

Simple finite element equation solver

## Installation

## One time setup

```console
git clone git@github.com:tjfulle/me7540
cd me7540
python3 -m venv venv
source venv/bin/activate
cd Exercises/fem
python3 -m pip install -e .
```

## Run an example

```console
cd examples
python3 plate_with_hole.py
```
